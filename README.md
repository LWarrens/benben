# benben
A game engine with these features:

- Node.JS scripting
- Vulkan renderer
- CEF GUI system
- npm & xmake package management

## Requirements
[Node.JS](https://nodejs.org/en/download)
and
[xmake + xrepo](https://xmake.io/#/getting_started)
must be installed and available on your terminal

## Build/Test/Running
### Build
Run `npm i; xmake;`
### Test/Run
Run `npm start`
